# WhatsApp Chatbot 🚀

A simple backend application built using Spring Boot that simulates a basic WhatsApp chatbot.

---

## 📌 Features
- REST Webhook API
- JSON request handling
- Basic chatbot response system

---

## 🛠 Tech Stack
- Java
- Spring Boot
- Maven

---

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse / IntelliJ
3. Run `WhatsappbotApplication.java`
4. Open Postman / Browser
